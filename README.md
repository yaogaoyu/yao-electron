# packager

-

使用本工具首先需要安装nodejs环境和npm环境。

使用 Typescript + React 作为开发库。

使用 Gulp 进行项目构建。

### 安装依赖包
`npm i`


### 更换打包出来的应用图标
替换icon目录下的文件即可

icon.icns OSX下的应用图标

icon.ico  Windows下的应用图标


### 重置打包工具
`gulp reset`


### 清除已打包
`gulp clean`


### 执行编译
`gulp bundle`


### 执行编译 调试模式
`gulp bundle:debug`


### 运行（开发环境）
`./bin/run`


### 打包
`gulp build`

打包请见var/.cache/README


### 打包后的文件存放目录
var/release
